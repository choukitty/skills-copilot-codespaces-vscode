Police

La propriété CSS qui permet d'indiquer la police à utiliser est font-family.

  • Exemples :
Code en CSS Code en HTML Aperçue 
			div {
				font-family : arial,georgia,verdana;
			}
  

Ceci est un paragraphe écrit avec la police Arial

Ceci est un paragraphe écrit avec la police Georgia

pour éviter les problèmes si l'internaute n'a pas la même police que vous, on précise en général plusieurs noms de police, séparés par des virgules

exemple : la police de l'élément <div> est Arial, sinon Georgia, sinon Verdana, sinon la police par défaut du navigateur.

div {
	font-family : arial,georgia,verdana;
}

Voici une liste de polices qui fonctionnent bien sur la plupart des navigateurs :

  • Arial
  • Arial Black
  • Courier New
  • Times New Roman
  • Comic Sans MS
  • Georgia
  • Verdana

Taille

La propriété CSS qui permet de modifier la taille du texte est font-size, la taille d'un texte peut être mesurée soit d'une manière absolue ou relative

  • Exemple :
Code en CSS Code en HTML Aperçue

p#taille_absolue{
font-size : 16px;
}

 

<p id="taille_absolue">
Ceci est un paragraphe de taille 16 pixels
</p>

 

Ceci est un paragraphe de taille 16 pixels

Ceci est un paragraphe une fois et demi plus grand que le texte actuel

Unité de mesure absolue (fixe):

  • px : pixel
  • mm : millimètre
  • cm : centimètre
  • in : pouce (2.54cm)
  • pt : point

Unités de mesure relative (par rapport à un autre élément)

  • % : relative à la taille du texte du parent
  • em : relative à la taille du texte actuel (pour grossir un texte utiliser une valeur >1, pour réduire un texte utiliser une valeur <1, pour un texte normale utiliser la valeur 1)
  • rem : relative à la taille du texte de la racine

Mise en forme de paragraphe

Alignement

La propriété CSS qui permet de changer l'alignement d'un paragraphe est text-align.

 

Les autres choix de l'alignment sont:

  • left : alignement à gauche
  • right : alignement à droite
  • justify : alignement en même temps à gauche et à droite

Direction

La propriété CSS à utiliser pour changer la direction d'un texte est direction

La direction du texte est par défaut de gauche à droite "ltr", si on a un texte écrir en arabe on doit changer sa direction de droite à gauche"rtl".

  • Exemple :
Code en CSS Code en HTML Aperçue
p#direction {
direction : rtl;
}		 <p id="direction">

Mise en forme de la bordure

Bordure

Pour tracer une bordure, il faut choisir :

  • un style de trait (border-style),
  • une épaisseur (border-size)
  • une couleur (border-color).

Si vous voulez changer l'arrondissement des coins, vous pouvez utiliser la propriété border-radius

Une bordure est composée de 4 traits

  • bordure du haut (top)
  • bordure de droite (right)
  • bordure de bas (bottom)
  • et bordure de gauche (left)

Pour préciser lequel des bordures à modifier, utiliser la propriété contenant la position de la bordure (border-top-style, border-top-size, border-top-color, border-left-style, ...)

  • Exemple: le titre de cette page possède une bordure simple (solid) de taille 2 pixels (2px) et de couleur rouge (red)
  • Exemple de bordures

border-style : solid; border-style : double; border-style : dotted;

border-style : dashed; border-style : inset; border-style : outset;
